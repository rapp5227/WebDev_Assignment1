Projects

  1. TommieCoin Wallet

    As a final project in Advanced Information Security, I created a command-line wallet application for a hypothetical non-distributed cryptocurrency called TommieCoin, created by Dr. Scott Yilek. This application included functionality for the server admin to create new currency, as well as for users to transact and mine their own TommieCoin.

  2. File Explorer

    As a final project in my Operating Systems course, I created a lightweight file explorer GUI with the guidance of Dr. Tommy Marrinan. It uses the SDL2 graphics library to display a directory's contents using a paging system, and users can click on these icons to navigate into them or open files using the system default application.

  3. Criminal Justice Capstone (Ongoing)

    As part of the capstone course for the Criminal Justice degree track at St. Thomas, I am currently participating in a service project benefitting the Women's Advocates domestic violence shelter in St. Paul. Specifically, I am working with a small team to develop a report on their aftercare program and resources that can be shared with other shelters to provide guidance for them to set up their own aftercare programs.
